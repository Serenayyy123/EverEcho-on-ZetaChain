# P0-F2 快速开始指南

## 🚀 5 分钟启动

### 1. 配置环境变量

```bash
cd frontend
cp .env.example .env
```

编辑 `.env`，填入合约地址：

```env
VITE_API_URL=http://localhost:3000
VITE_REGISTER_ADDRESS=0x...
VITE_EOCHO_TOKEN_ADDRESS=0x...
VITE_TASK_ESCROW_ADDRESS=0x...
VITE_CHAIN_ID=31337
VITE_NETWORK_NAME=Localhost
```

### 2. 启动前端

```bash
npm install
npm run dev
```

访问：http://localhost:5173

### 3. 测试流程

1. 连接 MetaMask 钱包
2. 访问任务广场：`/tasks`
3. 点击任务卡片查看详情
4. 根据角色测试操作按钮

## 🔧 故障排查

### 问题：任务列表为空
- 检查合约地址是否正确
- 检查网络是否匹配
- 确认是否有任务数据

### 问题：按钮不显示
- 检查钱包是否连接
- 确认当前地址角色
- 查看浏览器控制台错误

### 问题：交易失败
- 检查钱包余额
- 确认 EOCHO Token 授权
- 查看交易错误信息
